# Vertruvo

Vertruvo is now a build-ready SaaS starter instead of a product-specific shell.

## Stack

- Vite
- TypeScript

## Getting started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Customize this fork

Edit `src/main.ts` to update:

- app name
- headline and positioning
- feature pillars
- launch checklist

Edit `src/style.css` to change the visual brand.
